# angular-mimetype

Angular Mimetype is a module for search or match for [MIME](https://en.wikipedia.org/wiki/MIME) type 

### Requirements
- node
- npm
- gulp

### Build 
```sh
gulp 
```
for building 
```sh 
grunt sourcemaps
``` 
for build with source mapsi.
